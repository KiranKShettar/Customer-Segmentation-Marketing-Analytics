# Customer Segmentation & Marketing Analytics

## Project Overview

This project focuses on customer segmentation using machine learning techniques to identify distinct customer groups based on demographic and purchasing behavior. The analysis helps businesses understand customer characteristics, improve marketing strategies, and target customers more effectively.

## Dataset

* **File:** `marketing_campaign.csv`
* The dataset contains customer demographics, income, spending habits, purchase history, and campaign response information.

## Objectives

* Perform exploratory data analysis (EDA)
* Clean and preprocess the dataset
* Analyze customer demographics and purchasing behavior
* Build customer segments using K-Means Clustering
* Visualize customer groups and derive business insights

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Project Workflow

1. Data Loading
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Feature Scaling
6. K-Means Clustering
7. Cluster Visualization
8. Business Insights

## Key Insights

* Identified distinct customer segments based on income and spending patterns.
* Compared purchasing behavior across different customer groups.
* Analyzed demographic characteristics influencing customer value.
* Generated insights that can support personalized marketing campaigns.
